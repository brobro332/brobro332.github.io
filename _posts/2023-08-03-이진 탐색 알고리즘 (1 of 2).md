### 📌 개요
---

이진 탐색에 대해 알아보기 전에 가장 탐색 방법인 순차 탐색에 대해 먼저 이해할 필요가 있다. <br>
순차 탐색이란 리스트 안에 있는 특정한 데이터를 찾기 위해 앞에서부터 데이터를 하나씩 차례대로 확인하는 방법이다. <br>
리스트 자료형에서 count() 메소드를 이용할 때도 내부에서는 순차 탐색이 수행된다. <br>
순차 탐색의 시간 복잡도는 O(N)이다. <br>
<a style="color:red;">이진 탐색은 배열 내부의 데이터가 정렬되어 있어야만 사용할 수 있는 알고리즘이다.</a> <br>
이진 탐색은 변수 3개를 사용하는데, 탐색하고자 하는 범위의 시작점, 끝점, 그리고 중간점이다.<br>
이진 탐색의 시간 복잡도는 O(logN)이다. <br>
이진 탐색을 구현한느 방법에는 '재귀 함수'를 이용하는 방법과 단순하게 '반복문'을 이용하는 방법이 있다. <br>


#### 예제 소스 코드

```python
# 재귀 함수로 구현한 이진 탐색 소스코드
def binary_search(array, target, start, end):
    if start > end:
        return None
    mid = (start + end) // 2
    if array[mid] == target:
        return mid
    elif array[mid] > target:
        return binary_search(array, target, start, mid-1)
    else:
        return binary_search(array, target, mid+1, end)

n, target = list(map(int, input().split()))
array = list(map(int, input().split()))

result = binary_search(array, target, 0, n-1)

if result == None:
    print("원소가 존재하지 않습니다")
else:
    print(result+1)
```

```python
# 반복문으로 구현한 이진 탐색 소스코드
def binary_search(array, target, start, end):
    while start <= end:
        mid = (start + end) // 2 
        if array[mid] == target:
            return mid
        elif array[mid] > target:
            end = mid - 1
        else:
            start = mid + 1
    return None

n, target = list(map(int, input().split()))
array = list(map(int, input().split()))

result = binary_search(array, target, 0, n-1)

if result == None:
    print("원소가 존재하지 않습니다")
else:
    print(result+1)
```

#### 코딩 테스트에서의 이진 탐색

높은 난이도의 문제에서는 이진 탐색 알고리즘이 다른 알고리즘과 함께 사용되기도 한다. <br>
탐색 범위가 2,000만을 넘어가면 이진 탐색으로 문제에 접근하는 것을 권장한다. <br>
데이터의 개수나 값이 1,000만 단위 이상으로 넘어가면, <br> 
이진 탐색과 같이 O(logN)의 속도를 내는 알고리즘을 떠올려야 문제를 풀 수 있는 경우가 많다. <br>

### 💾 트리 자료구조
이진 탐색과는 조금 다르지만, 이진 탐색과 유사한 방법을 이용해 탐색을 항상 빠르게 수행할 수 있는 자료구조이다. <br>
<a style="color:red;">트리는 노드와 노드의 연결로 표현하며, 노드는 어떠한 정보를 가지고 있는 개체이다.</a> <br>
큰 데이터를 처리하는 소프트웨어는 대부분 데이터를 트리 자료구조로 저장해서 빠르게 탐색한다. <br>

#### 이진 탐색 트리
이진 탐색 트리란 이진 탐색이 동작할 수 있도록 고안된, 효율적인 탐색이 가능한 자료구조이다. <br>
다음은 찾는 원소가 37일 때 동작하는 과정이다.
    1️⃣ 루트 노드와 찾아야 할 원소값 '37'을 비교한다.
    2️⃣ 찾아야 할 원소가 더 크면 오른쪽 노드를 방문하고 더 작으면 왼쪽에 있는 노드를 방문한다.
    3️⃣ 위 과정을 반복하며 원소를 찾는다.

### ⚡ 빠르게 입력받기
데이터의 개수가 1,000만 개를 넘어가거나 탐색 범위의 크기가 1,000억 이상이라면 이진 탐색 알고리즘을 의심해보자. <br>
그런데 이렇게 입력 데이터가 많은 문제에 input() 함수를 사용하면 동작 속도가 느려서 시간 초과로 오답 판정을 받을 수 있다. <br>
이 때 sys 라이브러리의 readline() 함수를 이용하면 시간 초과를 피할 수 있다. <br>
```python
import sys

input_data = sys.stdin.readline().rstrip()

print(input_data)
```

sys 라이브러리를 사용할 때는 한 줄 입력받고 나서 rstrip() 함수를 꼭 호출해야한다. <br>
readline() 함수를 사용하면 줄의 끝에 개행 문자('\n')가 포함되는데 이를 제거해주어야 한다.
