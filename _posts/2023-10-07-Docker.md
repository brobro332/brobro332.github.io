## 💻 명령어

### 1️⃣ image 다운로드
- docker hub에서 명령어 복사
- docker pull 이미지명 (버전)

### 2️⃣ image 다운 및 실행
- docker run 이미지명
- -d: 백그라운드
- -p: 포트포워딩 
- -v: 폴더 연결
- <b> ❗ 데몬프로세스가 없을 때 </b>
  - docker run -dit ubuntu bash
  - docker attach 컨테이너ID (대화형 쉘 모드)
  - attach: bash 커맨드로 실행 중일 경우 터미널 모드로 전환하기 위한 명령어
- <b> ❗ 데몬프로세스가 존재할 때 </b>
  - docker run -d -p 8080:80 httpd
  - docker exec -it 컨테이너ID bash (대화형 쉘 모드)
  - bash 커맨드로 실행해버리면 해당 프로그램이 실행되지 않아서 프로세스가 바로 꺼져버림

### 3️⃣ 실행중인 프로세스 종료 
- docker ps 
- docker stop 컨테이너ID

### 4️⃣ 종료된 컨테이터 삭제 
- docker ps -a
- docker rm 컨테이너ID

### 5️⃣ 이미지 삭제
- docker images
- docker rmi 이미지명

### 6️⃣ 모든 이미지를 한번에 삭제: Linux/Mac/Windows - git bash
- docker stop $(docker ps -q)
- docker rm $(docker ps -a -q)
- docker rmi -f $(docker images -q)


### 7️⃣ 모든 이미지를 한번에 삭제: Windows
- for /f "delims=" %A in ('docker ps -q') do (set rm1=%A)
- for /f "delims=" %A in ('docker ps -a -q') do (set rm2=%A)
- for /f "delims=" %A in ('docker images -q') do (set rm3=%A)
- docker stop %rm1%
- docker rm %rm2%
- docker rmi -f %rm3%
