## ✅ 코딩 테스트 전 정리용

### 1️⃣ 라이브러리

- Counter: 요소의 갯수를 세는데 용이성과 시간복잡도 측면에서 적합
  - 

  ```python
  from collections import Counter

  nums = [1, 2, 3, 2, 1, 1]
  cnt_nums = Counter(nums)

  print(cnt_nums[1]) # 3 

  print(cnt_nums.most_common()[::-1]) # 3
  ```

- deque: popleft() 메서드가 용이

```python
from collections import deque

nums = [1, 2, 3, 4]
deque_nums = deque(nums)

deque_nums.popleft() # nums == [2, 3, 4]
```

- permutation: 순열 - 순서가 존재 

```python
from itertools import permutation

nums = [1, 2, 3]
temp = permutation(a, 2)

print(list(temp)) # temp == [(1,2),(1,3),(2,1),(2,3),(3,1),(3,2)]
```

- combination: 조합 - 순서가 존재하지 않음 == 중복을 허용하지 않음
```python
from itertools import combinations

nums = [1, 2, 3]
temp = combinations(a, 2)

print(list(temp)) # temp == [(1,2),(1,3),(2,3)]
```

### 2️⃣ 탐색

- DFS: stack 또는 재귀함수 이용
  - stack
    1. 문제에서 주어진 graph, root(시작 노드) 입력받기
    2. 방문 표시할 visited 리스트와 stack 리스트 초기화
    3. 스택에 root 노드 append()
    4. while 문 시작(stack에 원소가 하나라도 존재하는 동안 반복)
      5. stack.pop()
      6. 꺼낸 노드를 아직 방문하지 않았다면 visited에 append()
      7. stack.extend(graph[node])
  - 재귀함수
    - 
    
    ```python
    def dfs_recursive(graph, root, visited=[]):
    
    visited.append(root)
    
    for i in graph[root]:
        if i not in visited:
            dfs_recursive(graph, i, visited)
    
    return visited
    ```

- BFS: deque를 이용
  1. 문제에서 주어진 graph, root(시작 노드) 입력받기
  2. 방문 표시할 visited 리스트 초기화 
  3. while 문 시작(queue에 원소가 하나라도 존재하는 동안 반복)
    4. queue에서 한 노드를 꺼내기
    5. 꺼낸 노드를 아직 방문하지 않았다면 visited에 append()
    6. queue에 인접 노드 append()

  - 예제1 : x와 y가 배열의 행과 열이 거꾸로 된 개념이다보니 헷갈리는 점 주의하기
  
  ```python
  from collections import deque

  # bfs 함수 선언
  def bfs(x, y):
      global cnt
      q = deque([(y, x)])

      while q:
          y, x = q.popleft()

          for i in range(4):
              nx = x + directions[i][0]
              ny = y + directions[i][1]

              # 그래프의 범위를 벗어나면 continue
              if nx >= M or nx < 0 or ny >= N or ny < 0:
                  continue

              # 그래프 내에 위치하고
              # 아직 방문하지 않은 배추가 있다면
              if visited[ny][nx] == 0 and graph[ny][nx] == 1:
                  visited[ny][nx] = 1
                  q.append((ny, nx))

      return 1

  # 입력
  T = int(input())

  for i in range(1, T + 1):
      M, N, K = map(int, input().split())

      # 변수 초기화
      directions = [(-1, 0), (1, 0), (0, -1), (0, 1)]   # 상, 하, 좌, 우

      # 그래프 초기화
      graph = [[0] * (M) for _ in range(N)]
      for _ in range(K):
          X, Y = map(int, input().split())
          graph[Y][X] = 1

      visited = [[0] * (M) for _ in range(N)]

      cnt = 0
      for j in range(N):
          for l in range(M):
              if visited[j][l] == 0 and graph[j][l] == 1:
                  cnt += bfs(l, j)
      print(cnt)
    ```
  - 예제2: depth 구하기: BFS에서는 노드에 depth를 같이 저장해주어야 함
  
  ```python
  from collections import deque

  def bfs(x):
      q = deque([(x, 1)])
      visited[x] = 1
      depth = 1

      while q:
          item, depth = q.popleft()

          for i in graph[item]:

              if visited[i] == 0:
                  depths[i] = depth
                  visited[i] = 1
                  q.append((i, depth + 1))

  # 초기화
  N = int(input())
  start, target = map(int, input().split())
  M = int(input())
  graph = [[] for _ in range(N + 1)]
  for _ in range(M):
      a, b = map(int, input().split())
      graph[a].append(b)
      graph[b].append(a)
  depths = [0] * (N + 1)
  visited = [0] * (N + 1)

  bfs(start)

  result = depths[target]
  if depths[target] == 0 and visited[target] == 0:
      print(-1)
  else:
      print(result)
    ```
